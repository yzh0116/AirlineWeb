# AirlineWeb
本网站可用于虚拟航司或进行修改成为其他功能的网站，本网站地图使用高德地图，对于航司地图页面，我们使用主平台所提供的在线机组api，然后通过计算绘制出来的。This website can be used for virtual airlines or modified into other functional websites. The map of this website uses Amap. For the airline map page, we use the online crew API provided by the main platform and then draw it through calculation.
